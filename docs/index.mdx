# üêù LangSwarm

**Production-Grade Multi-Agent Orchestration**

LangSwarm is an async-first framework for building, orchestrating, and scaling AI agent swarms. It unifies **Model Context Protocol (MCP)**, enterprise memory (Postgres/BigQuery), and diverse LLM providers into a single, type-safe API.

```bash
pip install langswarm
```

## ‚ö° Quick Start

Build your first swarm in seconds.

<CodeGroup>

```python Simple Agent
import asyncio
from langswarm import create_agent

async def main():
    # 1. Create a context-aware agent
    agent = create_agent(
        model="gpt-4o",
        system_prompt="You are a senior python engineer."
    )
    
    # 2. Chat with streaming support
    response = await agent.chat("Explain async generators")
    print(response)

asyncio.run(main())
```

```python Swarm Orchestration
from langswarm import create_agent, register_agent
from langswarm.core.workflows import create_workflow, get_engine

# 1. Define specialized agents
researcher = create_agent("researcher", model="claude-3-5-sonnet", 
                         system_prompt="Gather comprehensive data.")
writer = create_agent("writer", model="gpt-4o", 
                     system_prompt="Synthesize research into a report.")

# 2. Register fleet
register_agent(researcher)
register_agent(writer)

# 3. seamless Handoff Workflow
workflow = create_workflow(
    name="Research Report",
    steps=[
        {"agent": "researcher", "task": "Research: {input}"},
        {"agent": "writer", "task": "Write report based on: {researcher.output}"}
    ]
)

# 4. Execute
result = await get_engine().run(workflow, input="Future of AI Agents")
```

</CodeGroup>

## üöÄ Key Capabilities

<CardGroup cols={2}>
  <Card title="MCP Native" icon="plug">
    First-class support for the **Model Context Protocol**. Connect agents to filesystem, GitHub, databases, and browser with zero glue code.
  </Card>
  <Card title="Provider Agnostic" icon="server">
    Switch models instantly. Support for **OpenAI, Anthropic, Gemini, Mistral**, and local models via generic interfaces.
  </Card>
  <Card title="Enterprise Memory" icon="database">
    Built-in persistence layers for **PostgreSQL (pgvector)**, **BigQuery**, **Redis**, and **ChromaDB**. State survives restarts.
  </Card>
  <Card title="Async First" icon="bolt">
    Built from the ground up for **asyncio**. Handle massive swarms and concurrent workflows without blocking.
  </Card>
