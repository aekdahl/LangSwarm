# âš¡ Quickstart

Build your first multi-agent system with LangSwarm.

<Steps>
  <Step title="Install LangSwarm">
    Install the package and set your API key.

    ```bash
    pip install langswarm openai
    export OPENAI_API_KEY="your-api-key-here"
    ```
  </Step>

  <Step title="Create Your First Agent">
    A simple "Hello World" to verify your setup.

    ```python
    import asyncio
    from langswarm import create_agent

    async def main():
        # Create a streaming-enabled agent
        agent = create_agent(model="gpt-4o", system_prompt="Be concise.")
        
        # Stream the response
        async for chunk in agent.chat_stream("Hello, LangSwarm!"):
            print(chunk.content, end="", flush=True)

    if __name__ == "__main__":
        asyncio.run(main())
    ```
  </Step>

  <Step title="Orchestrate a Swarm">
    The real power: make two agents collaborate.

    ```python
    from langswarm import create_agent, create_workflow
    from langswarm.core.agents import register_agent
    from langswarm.core.workflows import get_workflow_engine

    # 1. Specialized Agents
    researcher = create_agent("researcher", model="gpt-4o", 
                            system_prompt="You are a research analyst.")
    writer = create_agent("writer", model="gpt-4o", 
                        system_prompt="You are a technical writer.")

    # 2. Register
    register_agent(researcher)
    register_agent(writer)

    # 3. Workflow
    workflow = create_workflow(
        definition="Research Pipeline", # Utilizing simple_api workflow creation
        agents=[
             {"agent": "researcher", "task": "Find 3 key facts about: {input}"},
             {"agent": "writer", "task": "Summarize these facts: {researcher.output}"}
        ]
    )

    # 4. Run (Note: create_workflow returns a bound workflow that can just .run())
    print(await workflow.run(input_message="Quantum Computing"))
    ```
  </Step>
</Steps>
