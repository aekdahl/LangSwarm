## ðŸ”‹ Optional Extras

Unlock advanced capabilities with feature flags.

| Feature | Flag | Description |
| :--- | :--- | :--- |
| **Observability** | `[observability]` | Adds **LangFuse** integration for tracing. |
| **BigQuery Vector** | `[bigquery]` | Adds **Google BigQuery** vector search support. |
| **Telemetry** | `[opentelemetry]` | Adds **OpenTelemetry** exporters. |
| **Development** | `[dev]` | Adds testing and linting tools. |

## ðŸ”‘ Environment Setup

LangSwarm auto-detects API keys from environment variables.

```bash .env
# Required for your chosen provider
OPENAI_API_KEY="sk-..."
ANTHROPIC_API_KEY="sk-ant-..."
GOOGLE_API_KEY="AIza..."

# Optional: Observability
LANGFUSE_PUBLIC_KEY="pk-..."
LANGFUSE_SECRET_KEY="sk-..."
```
